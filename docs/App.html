Class: App

App(configs)

new App(configs)

Accepts configuration object with main app configs. Must contain fields for: container - query selector for the app container HTML element name - String name of the app data - data object with key-value pairs for the expected data structure of the app beforeStart - object with methods that should run before the app starts afterStart - object with methods that should run immediately after the app starts
