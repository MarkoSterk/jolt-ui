
## jolt-ui -- a simple framework for complex ideas

jolt-ui aims at helping developers create nice, responsive and fast UIs with minimal effort. The framework consists of three basic parts: App, Component and Router.

**The framework uses only vanilla JS so no third-party dependancies are required.** **For easier and faster templating Jolt-ui incorporates dot.Js engine. Please checkout their awesome work on: [https://olado.github.io/doT/index.html](https://olado.github.io/doT/index.html)**

If you are interested in this project and how to use it please check out our website: [https://jolt-ui.com](https://jolt-ui.com/)
